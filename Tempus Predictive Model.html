Separating the features and target and creating a split in the data which to train our models on will help us test the validity of our model as feature selection and modeling on the training set will not include bias from the testing data set. A split of 30% is chosen for the test set to leave enough training data to enrich the model while having a large enough testing set to tease out any of the model's shortcomings.

PCA will give us an unbiased representation of the relationship of features per observation when projected onto a 2d plane. clusting of points shows likeness of features which can predict response. LDA uses the targets to find a suitable projection to maximize separation of features based on class.

The cumulative variance also does not converge to 100% at 30 components. Feature selection is definitely required before creating a predictive model.

Ideally a pipeline testing the performance of each feature selection method and parameter in conjunction with the built model would work best however for simplicity sake the exploration will try a few feature selection methods on a few models.

The K-Best functions being used are univariate selection methods which will test relation of response against each feature and select the top most like features. The $\chi^2$ test works well for categorical features tested against a categorical response however for the sake of testing ANOVA is used (f_classif drops duplicate features and passes to f_oneway).

Errors being thrown are a result of not removing features which were all 0 or all 1.

One problem with kbest is dependance on the k value which will either leave in features which cause noise or remove important features for classification.

We will skip recursive feature elimination out of knowledge that we could most likely achieve a better selection through the use of LASSO regression and random forest classifiers.

def grid_clf_test(clf, param_grid, X_train,y_train,X_test,y_test,cv=10,scoring='accuracy'):
    grid = GridSearchCV(clf,param_grid=param_grid, cv=cv, scoring=scoring, n_jobs=-1)
    grid.fit(X_train,y_train)
    print('The best params: ', grid.best_params_)
    grid.predict(X_train)
    print('Prediction accuracy to training class: ',metrics.accuracy_score(y_train,grid.predict(X_train)))
    y_pred = grid.predict(X_test)
    print('Prediction accuracy to test class: ',metrics.accuracy_score(y_test,y_pred))
    print('Value Counts for Null prediction: \n',y_test.value_counts(normalize=True))
    print(metrics.confusion_matrix(y_test, y_pred))
    print(metrics.classification_report(y_test,y_pred))
    y_pred_probs = grid.predict_proba(X_test)[:,1]
    fpr, tpr, thresholds = metrics.roc_curve(y_test,y_pred_probs)
    plt.plot(fpr,tpr)
    plt.title('roc curve')
    plt.xlabel('False Positive')
    plt.ylabel('True Positive')
    print('The AUC is: {:.4f}'.format(metrics.roc_auc_score(y_test, y_pred_probs)))
